The Caribbean Sea

The Caribbean Sea was the first game i ever made, it is a 2d top down game like asteroids except the asteroid are ships that shoot at you. if a ship is destroyed it turns into a burnt ship. if the player goes out of the map, the game teleports the player to the other opposite side.

Project Crypt

Project Crypt was my third game that i created with a buddy for a group project. it is a top down dungeon like game where the object it to kill all enemies or collect all the coins to escape. project crypt is one of my favourites becuase it not only was coding it interesting, but my buddy for the group project was good at designing the characters.
